# ResumeFlow

ResumeFlow is a modern, responsive front-end project for a fictional resume builder platform. It demonstrates how a professional SaaS product can present its features while also providing dedicated authentication pages with a polished user interface.

## Overview

The project is built entirely using vanilla HTML, CSS, and JavaScript. It includes a complete marketing landing page along with separate Login and Sign Up pages, all following the same modern dark theme and responsive design.

## Features

### Landing Page
- Responsive navigation bar
- Hero section with email signup form
- Statistics section
- How It Works section
- Feature cards
- Resume template showcase
- Customer testimonials
- Call-to-action section
- FAQ section
- Professional footer

### Authentication Pages
- Separate Login page
- Separate Sign Up page
- Password visibility toggle
- Social login buttons (UI)
- Remember Me option
- Forgot Password link
- Responsive two-column authentication layout
- Modern illustration panel

## Tech Stack
